Image Storage Folder
====================

This folder contains product images organized by category.

Folder Structure:
- woodworking/  - Images for woodworking products
- engraving/    - Images for engraving products  
- crochet/      - Images for crochet products

Usage:
To reference an image in your code, use the path:
/images/woodworking/your-image.jpg
/images/engraving/your-image.jpg
/images/crochet/your-image.jpg

Note: Images placed in wwwroot are served directly by the web server.
Make sure image file names are web-safe (no spaces, use hyphens or underscores).
